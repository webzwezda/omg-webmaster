Document Object Model

Объектная модель документа

Дерево

Дерево состоит из:

  • Узлы - любой элемент дерева
    • Родительский - Элемент, из которого растет узел
    • Дочерние - Элементы, которые растут из узла
  • Корень - Элемент из которого растет дерево, элемент без родителей
  • Листья - элемент у которого нет потомков

Родители и потомки

Поиск родительского элемента:

console.log(document.parentElement);

Поиск дочернего элемента:

console.dir(document.children, 1);

Поиск по Css селектору:

console.log(document.querySelector("h1"));

Поиск всех элементов по селектору который возвращает коллекцию:

console.log(document.querySelectorAll(".top"));

Получение атрибутов и их редактирование

Для того что бы изменить значение атрибута:

var title1 = document.querySelector("h1");
title1.setAttribute('style', 'background: red');

Доступ к атрибуту:

console.log(title1.getAttribute('class'));

Изменение класса у объекта

var title1 = document.querySelector("h1");
console.log(title1.className);
title1.className = 'top-top';
console.log(title1.className);

classList

Имеет такие методы

  • add
  • remove
  • toggle - если нету добавит, если есть удалит
  • contains метод проверки присутствия класса в элементе, возвращает bool

toggle

elem.classList.toggle("you");
elem.classList.toggle("he");
console.log(elem.classList); //DOMTokenList ["example", "for", "ok", "understand", "he"]

Создание DOM элементов

У каждого DOM элемента есть два свойства отвечающие за его наполнение

Возвращает всю разметку:

console.log(description.innerHTML);

Возвращает только текстовое содержимое

console.log(description.textContent);

Вставка разметки

Вставка перед элементом body

document.body.insertAdjacentHTML("afterbegin", '<h1>insert</h1>'); // После открывающего тега
document.body.insertAdjacentHTML("beforebegin", '<h1>insert</h1>'); // Перед открывающим тегом body

Вставка после тега

document.body.insertAdjacentHTML("afterend", '<h1>insert</h1>'); // Вставка после закрывающего тега
document.body.insertAdjacentHTML("beforeend", '<h1>insert</h1>') // Вставка перед закрывающим тегом body
