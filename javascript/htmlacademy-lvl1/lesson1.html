Асинхронность

Асинхронность загружаемых скриптов поможет отключить блокировку загрузки контента страницы.

async

Позволяет загружать страницу во время загрузки скрипта и сам скрипт выполнится сразу после своей загрузки.

<script src="/js/scmain.js" async>

</script>
defer

Загружается асинхронно вместе с страницей и выполняется только тогда когда она полностью загрузится

<script src="/js/scmain.js" defer>

</script>

Он умеет работать с двумя типами данных.

  1. Приметивные
    1. Числа
    2. Текст (строковые)
    3. Логические значения (булевые)
    4. Служебные (null, undefined)
  2. Сложные

Про сложные типы данных рассказано на следующей лекции

Числа

Пара особенностей числового типа данных.

  • Поддержка дробных чисел которые записываются при помощи запятой 3.01
  • Infinity и Nan как числовые типы данных
  • Числа могут быть записаны в разных форматах 0xFF

Операторы

В языке три вида операторов

Унарные

Операторы, которые работают с одним значение

Бинарные

Операторы, которые работают с двумя значениями

Тернарные

Единственный в языке оператор который принимает на вход три значения

Типы данных операторов

  1. Математические
  2. Строковые операторы
  3. Логические операторы
    1. Сравнение
    2. Булевой логики
  4. Служебные операторы

Приведение типов данных

  • Приведение к числу
    • parseInt("10" , 10) === 10;
    • parseFloat("10.001") === 10.001
  • Приведение к строке
    • 10..toString()

Унарный оператор

Приводить к нужному типу данных можно при помощи унарного оператора

+"10" === 10;
-"3" === -3;

3 + "" === "3";

Функции

Способ повторного использования алгоритма.
И называть их будем англ глаголами.

var calculateDistance = function(time, speedOfFirstCyclist, speedOfSecondCyclist) {
    var distanceOfFirstCyclist = speedOfFirstCyclist * time;
    
    var distanceOfSecondCyclist = speedOfSecondCyclist * time;
    
    var totalDistance = distanceOfFirstCyclist + distanceOfSecondCyclist;
    
    return totalDistance;
};

var measureUnit = 'км';
console.log('Расстояние между первыми поселками: ' + calculateDistance(3, 12, 14) + measureUnit);
console.log('Расстояние между вторыми поселками: ' + calculateDistance(5, 1, 2) + measureUnit);
