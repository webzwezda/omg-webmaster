Циклы и рекурсия

Что бы поменьше использовать рекурсии мы будем создавать циклы

var washItem = function(item){
    item--;
    console.log(item);
    if(item > 0){washItem(item)}
};

washItem(5);

Циклы в js

while/do..while

Цикл с условием

var washItem = function(item){
   while(item-- > 0){
      console.log(item);
   }
};

washItem(5);
for

Цикл с начальным условием, условием выхода и законом изменения

var washItem = function(item){
    for (var i = 0; i < item.length; i++){
        console.log(item[i]);
    }
};

Сложные типы данных

Сложные типы данных

Это такие данные которые сложенны из нескольких простых значений

Массив

Упорядоченный набор элементов. Используется для описания группы значений, связанных между собой: обращение идем сразу ко всем значениям массива

Объект

Набор пар ключ-значение.

Массивы

При обращении к элементу массива мы можем использовать сложное выражение внутри литерала [] главное что бы выражение возвращало число.

var numbers = [1, 2, 3];

var counter =  numbers[Math.floor(Math.random() * numbers.length)];

console.log(counter + " " + "Округление вниз" + " " + "Поможет получить рандомное число из массива");

var numSign = numbers[Math.ceil(Math.random() * 2)];

console.log(numSign + " " + "Округление вверх");

var numRandom = Math.round(Math.random() * 2);

console.log(numRandom + " " + "Округленние до ближайшего целого");

Объекты

Создается при помощи {} (литерал объекта).

var cat = {};

Элементы объекта представляют собой набор пар ключ-значение, которые отделяются друг от друга двоеточием. Между собой пары разделяются запятыми.

var cat = {
  name: "Федя",  
  age: 4
};

Доступ к элементам объекта происходит через точку или квадратные скобки.

console.log(cat.name, cat["age"]);

Терминология объекта

Свойство

Все кроме функций, любые значения являются свойствами объекта.

Пример свойства объекта
var cat = {
  name: "Федя",  
  age: 4
};
Метод

Все функции записанные в объект называются методами.

var wizard = {
  name: "dadast",
  say: function () {
      console.log(wizard.name + " Ты все сольешь");
  }
};

console.log(wizard);
console.log(wizard.say);

API

Application public interface

Набор доступных свойств и методов для решения определенной задачи. Часто реализуется в виде объектов. Как правило, свойство задают параметры решения, а методы определяют действия.

Примеры
  • DOM - Работа с элементами страницы (в том числе и SVG)
  • BOM - Работа с браузером
  • Canvas - Создание графики
  • XMLHttpRequest - Отправка запроса на сервер
