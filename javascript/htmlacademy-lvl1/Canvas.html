Что такое Canvas

Имеет несколько “контекстов отрисовки” - способов работать с графикой. Самые популярные - обычная двумерная графика и Web Gl. Контекст представляет собой объект Js.

Web Gl

Рисовалка трехмерной графики

Canvas

низкоуровневая графика с большим количеством элементов, от которой требуется большая производительность или контроль над каждым пикселем изображения

  • Изменение параметров отрисовки производится через изменения свойств контекста
  • Отрисовка производится вызовом медота контекста

Когда лучше применять

  • Онлайн игры
  • Интерактивные визуализации
  • Графические эффекты для изображения
  • Работа с потоковым видео

Если в значении метода заменить 2d на webgl то мы получим отрисовку 3D графики

// DOM
var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");

// Градиент
var gradient = ctx.createLinearGradient(0, 0, 300, 150);
gradient.addColorStop(0, "green");
gradient.addColorStop(1, 'rgba(0, 255, 0, 0)');

// Заливка
ctx.fillStyle =  gradient;
ctx.fillRect(0, 0, 300, 150);

// Обвести
ctx.strokeStyle = "blue";
ctx.strokeRect(75, 0, 150, 150);

// Отчистка
ctx.clearRect(0, 0, 300, 150);
