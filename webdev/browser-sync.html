Живая разработка сайта на телефоне

У меня несколько usecase что бы использовать browser sync.

Установка

npm install -g browser-sync

Запуск из папки

Самый частый вариант использования это отслеживания всех файлов

browser-sync start --server --files "*.*"

Открытие сайта на телефоне

Что бы открыть наш проект на другом устройстве или должны находиться в одной wifi сети.

http://192.168.0.10:3000

В общем посмотрите в консоль и подставьте цифры, иногда открывается не тот порт который указан в примере.
