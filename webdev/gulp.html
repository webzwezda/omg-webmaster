Основы gulp

Со временем постараюсь разобрать как инициировать новый gulp проект и то как запустить старый.

  1. Установка
  2. Инициализация нового проекта

Установка

Нам потребуется

  1. npm
  2. gulp

Инициализация нового проекта

Для начала нам нужно инициализировать наш файл package.json если его нет он создастся и по идее если он есть он должен подтянуть все что в нем указанно но это не точно и будет проверенно позже.

npm init
npm i gulp --save-dev // инициализация содержимого package.json

После создания нового package.json нам понадобится инструмент который будет подтягивать новые версии пакетов которые использует наш package.json

npm install -g npm-check-updates

После мы можем использовать такие команды как

ncu 
ncu -a

Создание gulpfile.js

Используем команду

touch gulpfile.js

В наш gulpfile.js положим такое

 var gulp          = require('gulp'),
     rsync         = require('gulp-rsync');

Инициализация Gulp в самом проекте

sudo npm install gulp -D

Deploy

Для деплоя используем rsync

npm install gulp-rsync -D

Переустановка

Бывает так что глобально ставится 3.9 а локально 4.0 и нужно переустанавливать

sudo npm rm -g gulp
sudo npm install -g gulp-cli
