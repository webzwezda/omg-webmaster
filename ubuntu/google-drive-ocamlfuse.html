Установка и настройка Google Drive Ocamlfuse

Google Drive Ocamlfuse позволит получить ряд преимуществ перед гугл диском который работает через гном онлайн аккаунт.

Установка

Выполним эти две команды

sudo add-apt-repository ppa:alessandro-strada/ppa
sudo apt install google-drive-ocamlfuse

Настройка

Не пугайтесь отсутствия графического интерфейса

Создание папки

Создадим папку где будут храниться наши файлы с гугл диска

mkdir ~/Desktop/google-drive

Запуск

Что бы запустить приложение выполним команду

google-drive-ocamlfuse ~/Desktop/google-drive

Авто запуск

Добавим эту строку в Startup Applications

google-drive-ocamlfuse /home/ilya/Desktop/google-drive
