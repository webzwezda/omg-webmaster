Gnome Terminal

Установлен по умолчинию в убунту и мне жизненно необходимо накатить zsh

zsh

Для меня удобней пользоваться ей чем обычным bash

Установка

sudo apt-get install zsh

Теперь установим фреймворк Oh My ZSH!, есть два способа, с использованием cURL и wget, если cURL не установлен — используйте второй вариант:

curl -L http://install.ohmyz.sh | sh

Осталось указать системе какую именно обложку нужно использовать, меняем оболочку на ZSH (после ввода команды нужно будет ввести пароль администратора):

chsh -s /bin/zsh

Последний шаг — перезагрузка компьютера (подойдет любой способ):

sudo shutdown -r 0

Плагин Fasd

Утилита для быстрого перемещения по каталогам. Она запоминает все каталоги (и просто пути к файлам), которые ты использовал, и позволяет перемещаться между ними, указав лишь часть пути (даже несколько букв).

Требование одно: указанный тобой набор символов должен быть уникальным среди всех путей, которые запомнил fasd.

КомандаОписание
zперейти в каталог
zzпозволяет перейти в каталог, используя интерактивное меню (если введенный набор символов встречается в нескольких путях)
aпоказывает все сохраненные пути
vоткрывает файл в редакторе Vim

Установить в Ubuntu можно так:

sudo add-apt-repository ppa:aacebedo/fasd
sudo apt-get update
sudo apt-get install fasd

Далее необходимо активировать плагин fasd в ~/.zshrc:

plugins=(git fasd)

Установка tmux

Устанавливаю впервые на уже накатанный oh my zsh, заинтересовался tmux из за неудачного опыта с терминатором

sudo aptitude intall tmux

После установим запуск tmux по умолчанию, тоесть вместе с нашим zsh
Нужно добавить эту строчку в наш файл .zshrc

[[ $TERM != "screen" ]] && exec tmux

Shortcut

Немного информации здесь

Создаст новую сессию

tmux new -s bar -d

Подключиться к сессии по имени можно

bind + : attach -t session_name
