Установка кодеков

По умолчанию в Ubuntu кодеки не установлены, поскольку они имеют проприетарную лицензию. Но вы можете установить их вручную, для этого выполните команду:

sudo apt install ubuntu-restricted-extras libavcodec-extra libdvd-pkg

Настройка параметров

Отключение автоматической блокировки экрана

Отключение автоматической блокировки экрана. Выставим значение off.

Setting>Privacy>Screen Lock

Затухание экрана

Setting>Power>Block screen>Never
Dim screen when inactive>off
Automatic suspend>off

Сразу отключим переход в спящий режим при закрытии крышки

sudo nano /etc/systemd/logind.conf

Там заменим #HandleLidSwitch=suspend на HandleLidSwitch=ignore

Ночной режим Ubuntu

Монтирование второго диска

У меня есть второй ssd и его нужно смонтировать

Что есть на пк

Выводит все дивайсы и их разделы

sudo fdisk -l

Показ партишенов каждого диска

Так же можно увидеть что куда монтировано

lsblk

Программа для разбития и визуализации партов

Эта программа поможет увидеть откуда ноги у какого парта так же поможет создать таблицу или разбить

Программа так же показывает UUID Если на партишен нажать dump

sudo cfdisk /dev/sda

То же самое можно делать и через fdisk

Форматирование

sudo mkfs.ext4 -f /dev/sda1

После форматирования диск мантируется в /media/SSD папку мы создаем сами и даем ей права что бы писать туда могли все

sudo chmod 777 SSD/

Автоматическое монтирование при старте ситсемы

Нужно отредактировать файл /etc/fstab

sudo nano /etc/fstab

Туда вписываем это заклинание

/dev/sda1 /media/SSD  ext4 noatime,discard,lazytime,errors=remount-ro 0 0

И монтируем

sudo mount /media/SSD
