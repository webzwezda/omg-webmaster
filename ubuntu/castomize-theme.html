Кастомизация рабочего стола после установки

Кастомизируем рабочее пространство, будет немного похоже на Mac но это только в лучшую сторону

Gnome tweak tool

Первое что нам потребуется

sudo apt install gnome-tweak-tool

Gnome extension

Что бы устанавливать расширения из гугл хрома `

sudo apt install chrome-gnome-shell

Расширения можно посмотреть here

Нужно установить:

  • User Theme
  • Dash to Dock
  • Hide Activities
  • Clipboard indicator
  • Workspace indicator
  • KDE Connect
  • Remove Dropdown Arrows
  • Status Area Horizontal Spacing
  • Steal My Focus
  • World Time

World time

[

Где скачивать темы

Большая колекция и хорошая навигация gnome-look на нем много нужных нам расширений.

Выберем раздел Gnome 3 и скачаем понравившуюся тему, в видео ставилась тема gnome os

Я же ставил Mojave

Создадим папку для тем

mkdir ~/.themes

В эту папку и закидываем нашу тему, потом в приложении Gnome-Tweak-Tool изменяем

Icons

Иконки устанавливаются в папку .icons

mkdir ~/.icons

Раздел на сайте icons theme. Я установил такие темы иконок

  • Flat-Remix
  • Papirus

Planck and Dash to Dock

Самое вкусное оставил на последок. Воспользуемся установленным ранее расширением Dash to Dock

Откроем настройки и скрываем нашу панель

Position and size>Intelegent autohide

В открывшемся окне настроек Intelegent autohide отключаем самый первый тумблер и после этого наша панель будет видна только при просмотре всех приложений. Так же там выберем такую настройку dodge windows OFF

У меня была ошибка при которой было две панели и вылечилось это после отключение опции

Launch>Show Applications icon

Просто отключил тумблер и все окей

Plank

Установка

sudo apt install plank

Так же нужно установить в центре приложений plank preferences

Далее добавим приложение в автозагрузку.

Name: plank
Command: plank

Установка темы для Plank

Раздел на сайте Docks and Launchers>Plank Themes, я установил тему Mojave

Распаковываем тему и закидываем ее ~/.local/share/plank/themes

Я скачивал macOS Grey.zip

Conky manager

Взято с сайта

sudo apt install conky

Download Links:

conky-manager realpath

После того как все скачали ставим по очереди командой

# Первым будет realpath
sudo dpkg -i package_name

После того как все установили вводим команду

sudo apt -f install
